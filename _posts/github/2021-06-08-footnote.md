目前md的标准格式的脚注在github上无法显示，如
```
文章内容[^1]

[^1] 脚注1的说明
```
文章内容[^1]

[^1] 脚注1的说明

如要在github上添加脚注，可用以下方法：
```
文章内容sup id=a1[[1]](#f1)sup

span id=f11. [ ^](#a1)span 脚注1的说明
```
请注意，[ ^]里^左边有个空格，这十分关键
